How to build:
- install Maven
	- Windows: phoenixnap.com/kb/install-maven-windows
	- Ubuntu:  phoenixnap.com/kb/install-maven-on-ubuntu
	
- open terminal or cmd
- navigate to directory that contains src folder and pom.xml file
- in the terminal type the following command: 
	- mvn package
____________________________________________________________________
How to run:
- open terminal or cmd
- navigate to directory that contains src folder and pom.xml file
- in the terminal type the following command: 
	- mvn javafx:run
	
____________________________________________________________________
How to test:
- open terminal or cmd
- navigate to directory that contains src folder and pom.xml file
- in the terminal type the following command: 
	- mvn test
